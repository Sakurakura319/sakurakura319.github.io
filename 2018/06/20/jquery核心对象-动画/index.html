jquery核心对象--动画

2018/6/20

动画

基础

hide()

方法描述：隐藏元素。

​ 创建一个显示div，然后隐藏该元素

show()

方法描述：显示元素。

​ 创建一个隐藏div，然后显示该元素

toggle()

方法描述：如果隐藏就设置为显示，如果显示就设置为隐藏

​ 创建一个按钮，控制div显示和隐藏

渐变

fadeIn()

方法描述：通过淡入的方式显示匹配元素。

​ 创建一个按钮，控制div缓慢淡入

fadeOut()

方法描述：通过淡出的方式隐藏匹配元素。

​ 创建一个按钮，控制div缓慢淡出

fadeToggle()

方法描述：用淡入淡出动画显示或隐藏一个匹配元素。

​ 创建一个按钮，控制div淡入和淡出

滑动

slideDown()

方法描述：用滑动动画显示一个匹配元素。

​ 创建一个按钮，控制div滑动显示

slideUp()

方法描述：用滑动动画隐藏一个匹配元素。

​ 创建一个按钮，控制div滑动隐藏

slideToggle()

方法描述：用滑动动画显示或隐藏一个匹配元素。

​ 创建一个按钮，控制div滑动显示和滑动隐藏

自定义

animate()

方法描述：根据一组 CSS 属性，执行自定义动画，并且支持链式调用。

​ 创建一个div，默认div宽高100px，背景颜色为黑色，先让div宽度变为200px，再让div高度变为200px

stop()

方法描述：停止匹配元素当前正在运行的动画。
