bootstrap常用样式

2018/8/26

常用样式

标题

提供 .h1~.h6的样式名，标签设置对应的样式名则会有对应的标题效果

提供 .small的样式名，有副标题的效果。

段落

通过.lead 来突出强调内容(其作用就是增大文本字号，加粗文本，而且对行高和margin也做相应的处理。

强调

强调
.text-muted：提示，使用浅灰色（#999）
.text-primary：主要，使用蓝色（#428bca）
.text-success：成功，使用浅绿色(#3c763d)
.text-info：通知信息，使用浅蓝色（#31708f）
.text-warning：警告，使用黄色（#8a6d3b）
.text-danger：危险，使用褐色（#a94442）

对齐

对齐效果
Bootstrap通过定义四个类名来控制文本的对齐风格：
.text-left：左对齐
.text-center：居中对齐
.text-right：右对齐
.text-justify：两端对齐

关于列表

列表

  1. 去点列表
    class=“list-unstyled”

  2. 内联列表
    class=“list-inline”

  3. 定义列表
    使用样式 class=“dl-horizontal” 制作水平定义列表 : 当标题宽度超过160px时，将会显示三个省略号

基础样式
1）.table：基础表格

附加样式

.table-striped：斑马线表格

.table-bordered：带边框的表格

.table-hover：鼠标悬停高亮的表格

table-condensed：紧凑型表格，单元格没内距或者内距较其他表格的内距小

Bootstrap

一、Bootstrap是什么

Bootstrap诞生于2011年，来自Twitter公司，是目前最受欢迎的前端框架

是一个用于快速开发Web应用程序和网站的前端框架

Bootstrap是基于HTML、CSS、JS的，简洁灵活，使得Web开发更加快捷

概述：Bootstrap是一个建立一个页面，就可以在三个终端(PC端、平板、手机)上完美展示的响应式前端框架

二、为什么要用Bootstrap

因为Bootstrap是一个简洁、直观、强悍的前端开发框架，让web开发更迅速、更容易上手。

三、如何使用Bootstrap

1.下载Bootstrap库 https://v4.bootcss.com/

toggle()

fadeToggle()

方法描述：用淡入淡出动画显示或隐藏一个匹配元素。

​ 创建一个按钮，控制div淡入和淡出

滑动

slideDown()

方法描述：用滑动动画显示一个匹配元素。

slideUp()

方法描述：用滑动动画隐藏一个匹配元素。

slideToggle()

方法描述：用滑动动画显示或隐藏一个匹配元素。

jquery核心对象–遍历

遍历

each()

方法描述：遍历一个jQuery对象，为每个匹配元素执行一个函数。

​ 获取每一个li元素并把每一个li元素的标签及内容输出

筛选

first()

方法描述：获取匹配元素集合中第一个元素。
设置ul下第一个li的背景为红色

last()

方法描述：获取匹配元素集合中最后一个元素。

​ 设置ul下最后一个li的背景为红色

eq()

方法描述：获取匹配元素集合中指定位置索引的一个元素。

注： 索引下标从0开始

​ 设置ul下第二个li的背景为红色

not()

方法描述：从匹配的元素集合中移除指定的元素。

​ 设置ul下li标签的背景为红色，排除第二个li

filter()

方法描述：筛选元素集合中匹配表达式或通过传递函数测试的元素集合。

​ .filter(selector)

​ 类型：Selector

​ 一个用于匹配元素的选择器字符串

​ .filter(function(index))

​ 类型：function()

​ 一个函数用作测试集合中的每个元素。this是当前DOM元素

​ .filter(element)

​ element

​ 类型：element

​ 一个或多个DOM元素来匹配当前元素的集合。

​ .filter(jQuery object)

​ jquery object

​ 类型:object

​ 现有jquery对象，用于进一步筛选当前元素集合

jquery核心对象--事件

2018/6/11

jquery核心对象–事件

浏览器事件

resize()

方法描述：为 JavaScript 的 “resize” 事件绑定一个处理函数，或者触发元素上的该事件。

​ 当浏览器窗口的尺寸改变时，控制台输出“浏览器尺寸改变了”

scroll()

方法描述：为 JavaScript 的 “scroll” 事件绑定一个处理函数，或者触发元素上的该事件。

​ 当浏览器窗口的滚动条滚动时，控制台输出“浏览器滚动条改变了”

事件绑定

on()

方法描述：在选定的元素上绑定一个或多个事件处理函数。

​ 为按钮添加单击事件，当按钮单击的时候，向控制台输出“按钮被单击了”

off()

方法描述： 移除一个事件处理函数。

​ 为按钮添加单击事件，然后再解绑，这时候你在点击按钮看看是不是不会输出信息了

事件委托

delegate()

方法描述：设置事件委托。

​ 为ul下的所有li添加单击事件，要求将该单击事件委托给ul，当单击li时，所对应的li背景变为红色

undelegate()

方法描述：移除事件委托。

​ 要求移除上一节中设置的事件委托，然后在分别点击li进行验证是否移除事件委托

事件对象

event.currentTarget : 当前执行的DOM元素。

event.target : 触发事件的DOM元素。

event.pageX : 相对于页面的左上角。

event.pageY : 相对于页面的顶部。

event.clientX : 相对于视口的左上角。

event.clientY : 相对于视口的顶部。

event.offsetX : 相对于事件元素左上角。

event.offsetY : 相对于事件元素顶部。

event.key ： 键盘的按键信息。

event.preventDefault() ： 阻止事件默认行为。

event.stopPropagation() ： 防止事件向外冒泡。

表单事件

focus()

方法描述：当获取焦点时触发所绑定的函数。

​ 当文本框获取焦点时，设置其背景为红色

blur()

方法描述：当失去焦点时触发所绑定的函数。

​ 当文本框获取焦点时，设置其背景为红色，当文本框失去焦点时，设置其背景为绿色

change()

方法描述：当内容改变时触发所绑定的函数。

​ 当文本框内容改变时，就向控制台输出当前文本框的内容 \

select()

方法描述：当内容选择时触发所绑定的函数。

​ 当文本框的内容被选择时，就向控制台输出当前文本框的内容

submit()

方法描述：当表单提交时触发所绑定的函数。

​ 当表单提交的时候，弹出对话框“表单提交了” c

鼠标事件

click()

方法描述：当鼠标单击时调用所绑定的函数。

​ 为按钮绑定一个单击函数，然后点击按钮，在控制台输出“按钮被单击了”

dblclick()

方法描述：当鼠标双击时调用所绑定的函数。

​ 为按钮绑定一个双击函数，然后双击按钮，在控制台输出“按钮被单击了”

mousedown()

方法描述：当鼠标左键按下的时候调用所绑定的函数。

​ 当鼠标左键按下的时候，控制台输出“鼠标左键按下”

mouseup()

方法描述：当鼠标左键松开的时候调用所绑定的函数。

​ 当鼠标左键松开的时候，控制台输出“鼠标左键松开”

mouseenter()

方法描述：当鼠标进入目标元素的时候调用所绑定的函数。

​ 创建两个div，当鼠标移到外层div的时候，向控制台输出“mouse enter”

mouseleave()

方法描述：当鼠标离开目标元素的时候调用所绑定的函数。

​ 创建两个div，当鼠标移出外层div的时候，向控制台输出“mouse leave”

mouseover()

方法描述：当鼠标进入目标元素的时候调用所绑定的函数。

注：mouseenter事件和mouseover的不同之处是事件的冒泡的方式。mouseenter 事件只会在绑定它的元素上被调用，而不会在后代节点上被触发。

​ 创建两个div，当鼠标移到外层div的时候，向控制台输出“mouse over”，鼠标移到内层div的时候，向控制台输出“mouse over”，当鼠标移到外层div的时候，向控制台输出“mouse over”

mouseout()

方法描述：当鼠标离开目标元素的时候调用所绑定的函数。

注：mouseleave事件和mouseout的不同之处是事件的冒泡的方式。mouseleave 事件只会在绑定它的元素上被调用，而不会在后代节点上被触发。

​ 创建两个div，当鼠标移出外层div的时候，向控制台输出“mouse out”

mousemove()

方法描述：当鼠标指针在元素内移动时，mousemove事件就会被触发，任何HTML元素都可以接受此事件。

​ 鼠标在div内移动，获取当前鼠标相对div的位置坐标

hover()

方法描述：.hover()方法是同时绑定 mouseenter和 mouseleave事件。 

 **当鼠标进入div设置背景为红色，当鼠标移出div设置背景为绿色，默认背景为黑色**

键盘事件

keydown()

方法描述：当键盘按键按下的时候调用绑定的函数。

​ 当键盘按键按下的时候，输出当前的按键

keyup()

方法描述：当键盘按键松开的时候调用绑定的函数。

​ 当键盘按键松开的时候，输出当前的按键

keypress()

方法描述：keypress与keydown类似，当键盘按键按下的时候调用绑定的函数。

​ 当键盘按键按下的时候，输出当前的按键
