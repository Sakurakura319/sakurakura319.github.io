DOM内部插入

text()

方法描述：设置/获取元素的文本内容，该方法读写合一。

html()

方法描述：设置/获取元素的html内容，该方法读写合一。

append()

方法描述：向当前匹配的所有元素内部的最后面插入指定内容。 

     **为当前的ul向后添加一个列表项**
appendTo()

方法描述：将指定的内容追加到当前匹配的所有元素的最后面。

​ 为当前的ul向后添加一个列表项

prepend()

方法描述：向当前匹配的所有元素内部的最前面插入指定内容。

​ 为当前的ul向前添加一个列表项

prependTo()

方法描述：将指定的内容追加到当前匹配的所有元素的最前面。

​ 为当前的ul向前添加一个列表项

DOM外部插入

after()

方法描述：在匹配元素集合中的每个元素后面插入参数所指定的内容，作为其兄弟节点。 

 **在div的后边插入一个段落**
before()

方法描述：在匹配元素集合中的每个元素前边插入参数所指定的内容，作为其兄弟节点。

​ 在div的前边插入一个段落

insertAfter()

方法描述：.after()和.insertAfter() 实现同样的功能。主要的不同是语法，特别是插入内容和目标的位置。 对于 .after()，选择表达式在函数的前面，参数是将要插入的内容。对于 .insertAfter()，刚好相反，内容在方法前面，它将被放在参数里元素的后面

 **在div的后边插入一个段落**
insertBefore()

方法描述：.before()和.insertBefore()实现同样的功能。主要的不同是语法，特别是插入内容和目标的位置。 对于 .before()，选择表达式在函数前面，参数是将要插入的内容。对于 .insertBefore()，刚好相反，内容在方法前面，它将被放在参数里元素的前面。

​ 在div的前边插入一个段落

DOM移除

empty()

方法描述：删除所有匹配元素的子元素。

​ 将ul列表下所有的子节点全部移除

remove()

方法描述：删除所有匹配的元素。 

>注： 同时移除元素上的事件及 jQuery 数据

​ 将ul列表下所有的p子节点全部移除

DOM替换

replaceWith()

方法介绍：用提供的内容替换集合中所有匹配的元素并且返回被删除元素的集合。

​ 将ul下的所有li替换为p标签

replaceAll()

方法介绍：.replaceAll()和.replaceWith()功能类似，但是目标和源相反。

​ 将ul下的所有li替换为p标签

DOM拷贝

clone()

方法描述：创建一个匹配的元素集合的深度拷贝副本。如果传入一个true，则表示是否会复制元素上的事件处理函数，从jQuery 1.4开始，元素数据也会被复制。 

 **为ul列表创建一个深克隆并追加到body后**

DOM遍历

 **获取id为two元素的前边所有的兄弟元素并输出**
next()

方法描述：获取集合中每个匹配元素紧邻的后一个兄弟元素，可以提供一个可选的选择器来进行筛选。

​ 获取id为two元素的后边所有的兄弟元素并输出

Bootstrap

一、Bootstrap是什么

Bootstrap诞生于2011年，来自Twitter公司，是目前最受欢迎的前端框架

是一个用于快速开发Web应用程序和网站的前端框架

Bootstrap是基于HTML、CSS、JS的，简洁灵活，使得Web开发更加快捷

概述：Bootstrap是一个建立一个页面，就可以在三个终端(PC端、平板、手机)上完美展示的响应式前端框架

二、为什么要用Bootstrap

因为Bootstrap是一个简洁、直观、强悍的前端开发框架，让web开发更迅速、更容易上手。

三、如何使用Bootstrap

jquery核心对象--事件

jquery核心对象–事件

浏览器事件

resize()

方法描述：为 JavaScript 的 “resize” 事件绑定一个处理函数，或者触发元素上的该事件。

​ 当浏览器窗口的尺寸改变时，控制台输出“浏览器尺寸改变了”

scroll()

方法描述：为 JavaScript 的 “scroll” 事件绑定一个处理函数，或者触发元素上的该事件。

​ 当浏览器窗口的滚动条滚动时，控制台输出“浏览器滚动条改变了”

事件绑定

on()

方法描述：在选定的元素上绑定一个或多个事件处理函数。

​ 为按钮添加单击事件，当按钮单击的时候，向控制台输出“按钮被单击了”

off()

方法描述： 移除一个事件处理函数。

​ 为按钮添加单击事件，然后再解绑，这时候你在点击按钮看看是不是不会输出信息了

事件委托

delegate()

方法描述：设置事件委托。

​ 为ul下的所有li添加单击事件，要求将该单击事件委托给ul，当单击li时，所对应的li背景变为红色

undelegate()

方法描述：移除事件委托。

​ 要求移除上一节中设置的事件委托，然后在分别点击li进行验证是否移除事件委托

事件对象

event.currentTarget : 当前执行的DOM元素。

event.target : 触发事件的DOM元素。

event.pageX : 相对于页面的左上角。

event.pageY : 相对于页面的顶部。

event.clientX : 相对于视口的左上角。

event.clientY : 相对于视口的顶部。

event.offsetX : 相对于事件元素左上角。

event.offsetY : 相对于事件元素顶部。

event.key ： 键盘的按键信息。

event.preventDefault() ： 阻止事件默认行为。

event.stopPropagation() ： 防止事件向外冒泡。

表单事件

focus()

方法描述：当获取焦点时触发所绑定的函数。

​ 当文本框获取焦点时，设置其背景为红色

blur()

方法描述：当失去焦点时触发所绑定的函数。

​ 当文本框获取焦点时，设置其背景为红色，当文本框失去焦点时，设置其背景为绿色

change()

方法描述：当内容改变时触发所绑定的函数。

​ 当文本框内容改变时，就向控制台输出当前文本框的内容 \

select()

方法描述：当内容选择时触发所绑定的函数。

​ 当文本框的内容被选择时，就向控制台输出当前文本框的内容

submit()

方法描述：当表单提交时触发所绑定的函数。

​ 当表单提交的时候，弹出对话框“表单提交了” c

鼠标事件

click()

方法描述：当鼠标单击时调用所绑定的函数。

​ 为按钮绑定一个单击函数，然后点击按钮，在控制台输出“按钮被单击了”

dblclick()

方法描述：当鼠标双击时调用所绑定的函数。

​ 为按钮绑定一个双击函数，然后双击按钮，在控制台输出“按钮被单击了”

mousedown()

方法描述：当鼠标左键按下的时候调用所绑定的函数。

​ 当鼠标左键按下的时候，控制台输出“鼠标左键按下”

mouseup()

方法描述：当鼠标左键松开的时候调用所绑定的函数。

​ 当鼠标左键松开的时候，控制台输出“鼠标左键松开”

mouseenter()

方法描述：当鼠标进入目标元素的时候调用所绑定的函数。

​ 创建两个div，当鼠标移到外层div的时候，向控制台输出“mouse enter”

mouseleave()

方法描述：当鼠标离开目标元素的时候调用所绑定的函数。

​ 创建两个div，当鼠标移出外层div的时候，向控制台输出“mouse leave”

mouseover()

方法描述：当鼠标进入目标元素的时候调用所绑定的函数。

注：mouseenter事件和mouseover的不同之处是事件的冒泡的方式。mouseenter 事件只会在绑定它的元素上被调用，而不会在后代节点上被触发。

​ 创建两个div，当鼠标移到外层div的时候，向控制台输出“mouse over”，鼠标移到内层div的时候，向控制台输出“mouse over”，当鼠标移到外层div的时候，向控制台输出“mouse over”

mouseout()

方法描述：当鼠标离开目标元素的时候调用所绑定的函数。

注：mouseleave事件和mouseout的不同之处是事件的冒泡的方式。mouseleave 事件只会在绑定它的元素上被调用，而不会在后代节点上被触发。

​ 创建两个div，当鼠标移出外层div的时候，向控制台输出“mouse out”

mousemove()

方法描述：当鼠标指针在元素内移动时，mousemove事件就会被触发，任何HTML元素都可以接受此事件。

​ 鼠标在div内移动，获取当前鼠标相对div的位置坐标

hover()

方法描述：.hover()方法是同时绑定 mouseenter和 mouseleave事件。 

 **当鼠标进入div设置背景为红色，当鼠标移出div设置背景为绿色，默认背景为黑色**

键盘事件

keydown()

方法描述：当键盘按键按下的时候调用绑定的函数。

​ 当键盘按键按下的时候，输出当前的按键

keyup()

方法描述：当键盘按键松开的时候调用绑定的函数。

​ 当键盘按键松开的时候，输出当前的按键

keypress()

方法描述：keypress与keydown类似，当键盘按键按下的时候调用绑定的函数。

​ 当键盘按键按下的时候，输出当前的按键
